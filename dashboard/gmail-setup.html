Guía de Configuración Gmail — Inteligent Markets

CONECTAR GMAIL AL SISTEMA

Guía paso a paso para que Vale y Santi puedan enviar emails desde intelligentmarkets@gmail.com

PASO 01
VERIFICAR EN 2 PASOS (REQUISITO)
Google requiere que tengas la Verificación en 2 pasos activada para poder crear App Passwords. Si ya la tienes activa, pasa al Paso 2.
🔒 Ir a Seguridad de Google →
⚠️ Busca "Verificación en 2 pasos" y asegúrate que diga Activada. Sin esto no puedes continuar.
PASO 02
CREAR APP PASSWORD
Ve a la página de contraseñas de aplicaciones de Google. Aquí crearás una contraseña especial solo para IM System, diferente a tu contraseña normal de Gmail.
🔑 Crear App Password →
ℹ️ Si el link te da error 404, significa que la verificación en 2 pasos no está activa. Vuelve al Paso 1.
PASO 03
CONFIGURAR LA APP PASSWORD
En la página de App Passwords verás un formulario. Llénalo así:
📱 Seleccionar aplicación
Elige "Correo" o "Mail" del menú desplegable
💻 Seleccionar dispositivo
Elige "Otro (nombre personalizado)" y escribe "IM System"
✅ Haz clic en "Generar". Aparecerá una contraseña de 16 caracteres como: abcd efgh ijkl mnop
PASO 04
GUARDAR EN EL SISTEMA
Copia la contraseña de 16 caracteres que generó Google (con o sin espacios, funciona igual) y agrégala al archivo .env de IM System.
Opción A Si instalaste con install.py — el archivo .env ya existe
# Abre el archivo .env en tu carpeta im-system/
nano ~/im-system/.env

# Busca esta línea y pega tu App Password:
IM_EMAIL_PASSWORD=abcdefghijklmnop

# Guarda con Ctrl+O → Enter → Ctrl+X
Opción B Exportar en terminal para sesión actual
# Pega en tu terminal (reemplaza con tu contraseña real)
export IM_EMAIL_PASSWORD="abcd efgh ijkl mnop"

# Para que sea permanente, agrégalo a tu perfil de shell:
echo 'export IM_EMAIL_PASSWORD="tu-password"' >> ~/.zshrc
# O si usas bash:
echo 'export IM_EMAIL_PASSWORD="tu-password"' >> ~/.bashrc
PASO 05
VERIFICAR QUE FUNCIONA
Prueba que la conexión funciona correctamente antes de enviar emails reales. El modo interactivo genera y envía un email de prueba.
# Ve a tu carpeta de instalación
cd ~/im-system

# Prueba a Vale en modo interactivo
# Ingresa TU propio email como destinatario para hacer la prueba
python agent/im_agents.py --agente valentina --interactivo

# Si ves "✅ Enviado", todo está configurado correctamente
✅ Cuando veas "✅ Enviado" en el terminal, ¡estás listo para prospectar!
⚠️ Si ves error "SMTPAuthenticationError": la App Password está mal copiada. Vuelve al Paso 3 y crea una nueva.
IMPORTANTE
LÍMITES DE GMAIL
Gmail tiene límites de envío. Respétalos para mantener la cuenta saludable:
📊 Límite diario
500 emails/día con Gmail estándar.
Los agentes ya incluyen pausas de 60-120 seg entre emails.
✅ Recomendación IM
Máximo 40-60 emails/día para mantener la reputación del dominio y evitar spam.
💡 Si necesitas volumen mayor, considera Google Workspace (límite de 2,000/día) o SendGrid con dominio propio.
OPCIONAL
CONECTAR APOLLO.IO (MEJORA LEADS)
Apollo.io permite encontrar emails verificados de decisores. El plan gratis da 50 contactos/mes.
Crear cuenta Apollo.io →
# 1. Ve a: app.apollo.io/settings/api-keys
# 2. Crea una nueva API Key
# 3. Agrégala al .env:
APOLLO_API_KEY=tu-api-key-aqui

# 4. Úsala en las búsquedas con --apollo:
python agent/lead_finder_v2.py --nicho odontologos --city Medellín --country Colombia --apollo
🆓 Plan Gratis
50 emails verificados/mes. Perfecto para empezar.
💰 Plan Basic ($49/mes)
1,000 emails/mes. Recomendado para IM cuando el volumen crezca.
